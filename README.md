# About Morpheus
Morpheus is a simple code editor built for people who do not want to wase their time. Less distraction, more coding. With a light theme and a simple UI, it is design for efficency. And on top of that, it is Open-source.

It works ***online*** and ***offline***. Meaning you can access it at any time if you have connection or download it to use it even without being connected.

# How to use it
Download the repo as a `.zip` file, unzip it and run the `Morpheus.html` file.
